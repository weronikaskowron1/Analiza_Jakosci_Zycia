# Analiza jakości życia

Eksploracyjna analiza danych dotyczących zdrowia, snu i stylu życia, z wizualizacjami pokazującymi zależności między stresem, snem, aktywnością fizyczną, BMI, wiekiem i zawodem.  

## Technologie  
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  

## Zakres prac  
- Wczytanie i czyszczenie danych (braki danych, transformacje zmiennych)  
- Podział wieku na kategorie i analiza grupowa  
- Analiza statystyczna zmiennych numerycznych  
- Badanie zależności między stresem a jakością snu i tętnem  
- Analiza wpływu BMI na ciśnienie krwi  
- Porównanie kobiet i mężczyzn pod względem snu, stresu i aktywności  
- Wizualizacja danych (heatmapa korelacji, wykresy słupkowe, scatterploty, wykres kołowy)
